In-Game Settings

Hotkey Reminder

Show an overlay hotkey reminder upon launching Omega Strikers.

Hotkey: Show / Hide In-Game Overlay

Assign a hotkey to toggle the in-game overlay.

Secondary Monitor

Show the desktop app on your second screen while the game is open.

Streamer Mode

Hide nicknames of your teammates and opponents anywhere they appear in the app.

App Settings

Auto-Launch

Automatically open this app when Omega Strikers launches.

Awakening Infobox

Display an awakening's information when hovering over its icon.

Default Striker Portraits

Set portrait style and size used to display strikers.
This will not overwrite any individually selected default or custom upload.
Large portraits can be fully viewed by expanding the app window vertically.

Linked Account

Set the default Omega Strikers account to obtain and store data for.
Leave blank to disable any default.

Storage & Data Settings

Export Match History

Export your match history to a file.

Load Match History

Load match history from a file. This will replace your current history.

Reset Match History

Wipe current match history data to a blank state.

Reset Settings

Restore the default settings to initial installation. This will not affect your match history.
